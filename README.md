🖤 Tenue Style – Streetwear E-commerce Website

Tenue Style is a modern streetwear e-commerce website built with React. It features a premium dark design, smooth animations, and a clean user experience inspired by top fashion brands.

✨ Features
Dark, minimal, and modern UI
Fully responsive (mobile, tablet, desktop)
Product categories: T-shirts, Hoodies, Jeans, Shoes
Shopping cart with total price calculation
Checkout system with delivery form
Delivery pricing:
Casablanca: 25 MAD
Outside Casablanca: 40 MAD
Smooth animations using Framer Motion
Sticky navbar
WhatsApp floating button
Contact form
🧠 Technologies Used
React.js
Tailwind CSS
JavaScript (ES6+)
Framer Motion
🚀 Getting Started
1. Clone the repository
git clone https://github.com/your-username/tenue-style.git
cd tenue-style
2. Install dependencies
npm install
3. Run the project
npm run dev

🎯 Purpose

This project was created to simulate a real-world fashion e-commerce website with a premium look and feel. It can be extended with backend integration, payment systems, and an admin dashboard.
