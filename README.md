## ✨ About the Project  
Tenue Style is a modern streetwear e-commerce platform built with React.  
It features a premium dark design and smooth user experience inspired by top fashion brands.

---

## 🧩 Features  
- Dark, minimal, premium UI  
- Fully responsive design  
- Product showcase (T-shirts, Hoodies, Jeans, Shoes)  
- Shopping cart system  
- Total price calculation  
- Checkout with delivery form  
- Delivery pricing system  
- Smooth animations (Framer Motion)  
- Sticky navbar  
- WhatsApp button  
- Contact form  

---

## 🧠 Tech Stack  
- React.js  
- Tailwind CSS  
- JavaScript (ES6+)  
- Framer Motion
